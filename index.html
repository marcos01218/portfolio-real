Biblioteca Robótica: Uma Webpage Personalizada

O projeto é uma página web criada com HTML e CSS, destinada a exibir uma coleção pessoal de projetos de robotica. O objetivo é criar um ambiente virtual onde seja possível compartilhar seus projetos preferidos.

A estrutura do site é baseada em HTML, que define a semântica e o layout do conteúdo, enquanto o CSS é usado para estilizar a página visualmente, incluindo cores, tipografia e a disposição dos elementos.

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido por uma divisão principal (.container) que apresenta o propósito do site e a coleção de projetos em uma seção flexível (.projetos). Cada projeto é destacado em seu próprio contêiner (.projeto), mostrando uma imagem da capa. O design responsivo é garantido pelo uso de uma meta tag viewport e um layout flexível que se adapta a diferentes tamanhos de tela.

O CSS personaliza o visual do site, usando variáveis para cores, estilizando o texto com uma fonte importada do Google Fonts e aplicando um esquema de cores suaves e botões interativos. O uso de HTML e CSS é importante pois assim é possível criar um site acessível e esteticamente agradável sem a necessidade de scripts complexos, com foco na usabilidade e na experiência da pessoa usuária. A escolha da tipografia e do esquema de cores contribui para a atmosfera acolhedora do site, incentivando a exploração da coleção de projetos.

Barbearia que não é do alura

O objetivo é engajar os espectadores em um cenário fictício que aborda o avanço dos cortes de cabelo e suas possíveis consequências para a humanidade, destacando a importância da reflexão ética sobre os cortes do mal.

O HTML estrutura o conteúdo da página, dividindo-o em seções como a tela inicial, cortes, preços e etc. O CSS é utilizado para estilizar a página, empregando um esquema de cores futurista e layout responsivo para melhorar a experiência do usuário.

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido por uma divisão principal (.container) que apresenta o propósito do site e a coleção de cortes em uma seção flexível (.cortes). Cada corte é destacado em seu próprio contêiner (.corte), mostrando uma imagem da capa, detalhes do corte e um link de compra. O design responsivo é garantido pelo uso de uma meta tag viewport e um layout flexível que se adapta a diferentes tamanhos de tela.
